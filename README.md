# NSR-SSH

> **NSR:** No Subscription Required

An SSH client with a built-in vault. Save your hosts, personalize your workspace, and connect seamlessly.

It's a shame that you need to pay $15/month to use a good SSH client—often without new features or updates—just to get "Cloud Synchronization". So, use this project, modify it, and do whatever the f*ck you want with it!

## Availability

Built with **Rust**. Currently running on **Linux** (Windows and macOS support planned for the future).
